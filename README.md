# khuthon-backend

본 레포지토리는 경희대학교 중앙 IT 동아리 쿠러그에서 주관하는 해커톤 대회인 khuthon의 백엔드입니다.

## Prerequirements

- Node.js v20.11.1
- Docker

## Installation

```shell
git clone https://github.com/khu-khlug/khuthon-backend.git
npm install
```

## Run

```shell
docker compose up
```
